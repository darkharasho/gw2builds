Shimmer animation style

Which loading animation feels right for the dark UI theme?

A. Sweep Shimmer

Classic Facebook-style left-to-right light sweep across dark placeholder bars. Smooth, familiar.

B. Pulse Glow

Subtle opacity pulsing. Less motion, more ambient. Feels calmer on dark backgrounds.

C. Slow Sweep (Wider Highlight)

Same sweep pattern but slower, with a wider/softer highlight band. More polished feel.
