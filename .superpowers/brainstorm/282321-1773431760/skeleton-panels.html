Which panels get skeleton loading?

Select all the areas that should show shimmer placeholders while loading. The app has these main visual areas:

1

Skill Bar

The row of heal / utility / elite skill slots at the top of the Build tab. Could show ~5 circular or rounded-square placeholders.

2

Specializations Panel

The 3 spec lines with trait trees. Could show 3 rows of rectangular placeholders mimicking the spec selector + trait grid structure.

3

Equipment Panel

Armor, weapons, trinkets, food slots in the Equipment tab. Could show a grid of square slot placeholders.

4

Detail / Reference Panel

The right-side panel that shows skill/trait details. Usually empty until you click something — skeleton here might be misleading.

5

Build Library List

The list of saved build cards on the Library page. Could show 3-4 card-shaped placeholders.

6

Toolbar (Profession Dropdown)

The profession selector dropdown in the toolbar. Could show a placeholder select box while professions load.
