When should skeleton loading appear?

The app has a ~1-3 second blank window while GW2 API data loads. When should we show shimmer placeholders?

A

First Launch Only

Show skeletons only on initial app load. When switching professions mid-session, the old content stays visible until the new data arrives (feels snappier since there's already content on screen).

B

Every Data Load

Show skeletons any time panels are waiting for data — initial launch, profession switches, game mode toggles. Consistent loading experience but means brief skeleton flashes even on warm cache (~50ms).

C

Smart — Only When Empty

Show skeletons only when a panel has no content yet. If switching professions, keep the old content visible until new data replaces it. Skeletons only appear for truly blank panels (first load, or after creating a new build with no profession).
